RAM Monitor - Windows System Tray Utility
=========================================

QUICK START
-----------
1. Double-click "ram-monitor.exe"
2. Look for a white square icon in your system tray (bottom-right, near the clock)
3. Hover over the icon to see live stats

TOOLTIP FORMAT
--------------
Example: "3.44/16.0 RAM  67%CPU"
         ^^^^       ^^^^
         Used RAM   CPU Usage
              ^^^^
              Total RAM

- RAM shown in GB (Gigabytes)
- CPU shown as percentage (0-100%)
- Updates every second

STOPPING THE APP
----------------
- Press Ctrl+C in the terminal window, OR
- Use Task Manager to end "ram-monitor.exe"

FILE SIZE
---------
Only 184 KB! Minimal memory usage (<5MB RAM)

REQUIREMENTS
------------
- Windows 10 or later
- No installation needed
- No administrator rights required

TROUBLESHOOTING
---------------
Q: I don't see the icon?
A: Click the ^ arrow in the system tray to show hidden icons

Q: CPU shows 0%?
A: Wait 1-2 seconds for the first calculation cycle

Q: How do I make it start with Windows?
A: See the main README.md in the repository for auto-start instructions
